# 異常観測ゲーム（Anomaly Observation）

日常に紛れ込んだ「異常記録」を読み、
その異常度を数値で判定していく観測型ゲーム。

恐怖は直接描写せず、
記録・数値・演出によって違和感が蓄積していく構成。

## ゲーム内容
- プレイヤーは記録員として異常記録を確認
- 異常度を数値で判定
- 判定結果は一覧・分析ページで確認可能
- 済スタンプや画面演出によって進行を可視化
- 
## 工夫した点
- 数値と統計で「異常」を表現する設計
- ホラーを直接描かず、演出で違和感を生む構成
- 判定履歴を分析ページで可視化

## 起動方法
1. 仮想環境を有効化
2. 必要なライブラリをインストール
3. `python manage.py runserver`

※ 学習・制作発表用に作成した作品です。
